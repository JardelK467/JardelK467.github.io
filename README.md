# Jardel Portfolio

Simple static portfolio website built with plain HTML and CSS.

## Files
- index.html: page structure and content
- style.css: visual theme, layout, and responsive behavior

## Run locally
1. Open index.html directly in your browser, or
2. Serve with a local server for a cleaner workflow:

```bash
cd /Users/jardelkerr/jardel-portfolio
python3 -m http.server 8080
```

Then open http://localhost:8080.

## Quick customization
- Replace the headshot image URL in index.html.
- Update project links and descriptions.
- Edit books and music lists.
- Change contact email + social links.
